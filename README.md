This project, developed during a professional internship at Codec Technologies, addresses the critical challenge of information filtering in the digital era. As content libraries grow exponentially, personalized discovery is essential for user retention. This work implements a Hybrid Recommendation System designed to overcome the limitations of traditional single-model architectures, specifically the "Cold Start" problem and "Popularity Bias."
The system is built on a two-tier architecture. The first tier utilizes Content-Based Filtering through TF-IDF (Term Frequency-Inverse Document Frequency) vectorization and Cosine Similarity to map item-to-item relationships based on metadata. The second tier implements Collaborative Filtering using a K-Nearest Neighbors (KNN) algorithm and Matrix Factorization to identify patterns in user behavior and ratings. To enhance real-world usability, a Fuzzy Logic string-matching layer was integrated to handle natural language search queries and typographical errors.
Data engineering was performed on the MovieLens 100k dataset, utilizing Compressed Sparse Row (CSR) matrices to optimize memory efficiency during the training of the collaborative model. Evaluation was conducted using Root Mean Square Error (RMSE) to measure predictive accuracy. The results demonstrate that the hybrid approach provides significantly more diverse and relevant recommendations compared to standalone models. This internship project not only demonstrates industry-level implementation of AI but also provides a foundational framework for the author’s ongoing research into Graph Attention Networks (GATs) for advanced recommendation logic.
